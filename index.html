Figissimo!

Story

This project was a collaboration with mareculturaleurbano.
Figino is a small town next to Milano and one of the first places, in the area of Milano, with a cohousing project.
The idea of this project was to create social connection between the old citizens and the newest one.
For do that mareculturale has created a game where the citizens must tell an anecdote regarding the city and than nominate another citizen that has to partecipate.

Scope

Their need was to create a platform for visualize this long narration and that allow the citizens to keep it updated indipendently.

Result

The final product was an online platform with a custom map that show all the connections between all citizens, with the possibility to add new ones.

You can visit here the platform

My Role

In the early stage collaborate with the art director for realize the concept.

Choose the right tools for develop the idea: leaflet js, javascript and wordpress.

Collaborate with the graphic designer.

Create the frontend and the backend

Fornace della memoria

Concept

This project was a collaboration with Daniele Fadda

The goal was to find an interesting way to show the 16 interview realized for the temporary exhibition "La fornace della meoria".
We create an interacttive table ,using processing and the tuio protocol, with a tangible interface

How dose it work

Inside the table there is a projector and a PC with a webcam.
To each cube is assciated a video and a fiducial marker that make each cube unique.
A custom software made it with processing and the tuio protocol read the data from the webcam and check which video play.

My Role

In the early stage elaborate the concept.

Choose the right tools for develop the idea: processing, tuio protocol

Create the interface

Walk on the digital side

Concept

This project was a collaboration with the computer vision lab of Politectico di Modena

The goal was to find a different way to use an interactive floor realized by the lab
We create an interacttive installation, where the user waliking culd eplore remote area of the earth

How dose it work

The interactive flor can detect the position of one person in his area.
The data generated by the floor are sended to a web page through a server.
A custom software based on Hyperlapse js and javascript read the data and generate the envirorment around him.
