Figissimo!

Story

This project was a collaboration with mareculturaleurbano.
Figino is a small town next to Milano and one of the first places, in the area of Milano, with a cohousing project.
The idea of this project was to create social connection between the old citizens and the newest one.
For do that mareculturale has created a game where the citizens must tell an anecdote regarding the city and than nominate another citizen that has to partecipate.

Scope

Their need was to create a platform for visualize this long narration and that allow the citizens to keep it updated indipendently.

Result

The final product was an online platform with a custom map that show all the connections between all citizens, with the possibility to add new ones.

You can visit here the platform

Fornace della memoria

Concept

This project was a collaboration with Daniele Fadda

The goal was to find an interesting way to show the 16 interview realized for the temporary exhibition "La fornace della memoria".
We have created an interactive table with a tangible interface, using Processing and the "tuio" protocol.

How does it work

Inside the table there is a projector and a PC with a webcam.
To each cube is associated a video and a fiducial marker that make each cube unique.
The custom software is made with processing and the "tuio" protocol that read the data from the webcam and set which video to play.

My Role

Develop the concept.

Choose the right tools for develop the idea: processing, tuio protocol.

Create the interface.

Create the software.

Walk on the digital side

Concept

This project was a collaboration with the computer vision lab of Politectico di Modena.

The goal was to find a different way to use an interactive floor realized by the lab.
We have created an interactive installation, where the user while walking can explore remote areas of the earth.

How does it work

The interactive floor can detect the position of one person in its area.
The data generated by the floor are sent to a web page by the server.
A custom software based on Hyperlapse.js and javascript read the data and generate the environment around the person walking through the installation.

Elaborate the concept with the lab.

Choose the right tools to develop the idea.

Develop the interface.

Pop the Movie

Concept

This is a project that i have develop with Carina Lindmeier for the Ars Elctronica festival.
Popcorn is an audio-visual installation with the aim of making real popcorn by performing a composition based on "The Arrival of the Train" â€“ the first movie ever shown â€“ and the sounds of the biggest and most famous movie companies. This installation shows the arbitrary relation between the corn, the popped corn, movies and music itself and creates a unique position and a new view for the attendant. We question the relationship between art and reality but also our perception of the world of cinema and its significant value.

How does it work

The installation is based on a popcorn machine which activates the movies. Every time a popcorn pops up the movie contiunes with the next frame. So every single popcorn has an immediate effect on the whole process of the projected movie. To emphasize the process and the relationship with the movies nowadays sounds from big movie companies were added to the installation. The movie will be projected from the top onto the prepared popcorn.

My Role

Elaborate the concept.

Choose the right tools to develop the idea: processing, arduino.

Create the interface.

Develop the software.
